О проекте

Unified Network — это современное веб-приложение для управления ИТ-активами организации. Цель проекта — предоставить удобный и эффективный инструмент для учёта, мониторинга и управления устройствами, программным обеспечением и сетевой инфраструктурой.

Команда

Над проектом работал студент, специализирующийся на разработке программного обеспечения и управлении ИТ-инфраструктурой.

  • Руководитель проекта: Руслан Кутараев
  • Разработчик: Студент

Технологии

В проекте используются современные технологии и инструменты:

  • Frontend: HTML, CSS, JavaScript
  • Backend: Java, Spring Boot
  • База данных: MySQL
  • Инструменты: Maven, Git, Docker
