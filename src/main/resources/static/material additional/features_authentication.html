Возможности Unified Network

Учёт устройств

Unified Network позволяет добавлять и классифицировать устройства по типам: серверы, рабочие станции, принтеры, маршрутизаторы и коммутаторы. Каждое устройство регистрируется по IP-адресу, MAC-адресу и другим уникальным параметрам.

Добавление устройства

Заполните форму с основными параметрами: имя устройства, IP-адрес, MAC-адрес и тип устройства.

Подтверждение

Сохраните устройство кнопкой "Добавить устройство".

Мониторинг статуса

Отслеживайте состояние устройств (онлайн/офлайн), загрузку процессора, использование памяти и дискового пространства. Система предоставляет актуальные данные о каждом устройстве в реальном времени.

Загрузка процессора

Просматривайте текущую загрузку процессора каждого устройства.

Использование памяти

Отслеживайте использование оперативной памяти.

Дисковое пространство

Контролируйте использование дискового пространства.

Учёт программного обеспечения

Ведение списка установленного программного обеспечения для серверов и рабочих станций. Вы можете управлять версиями, издателями и датами установки каждого приложения.

Список ПО

Просматривайте список установленного программного обеспечения на каждом устройстве.

Сетевая карта

Визуализация расположения устройств и их связей в корпоративной сети. Вы можете добавлять устройства на карту, создавать связи между ними и сохранять текущую конфигурацию.

Добавление устройств

Добавьте устройства на сетевую карту.

Создание связей

Создайте связи между устройствами.

Сохранение конфигурации

Сохраните текущую конфигурацию сетевой карты.
